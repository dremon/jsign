Class PESigner

java.lang.Object
  net.jsign.PESigner

public class PESigner
extends Object

Sign a portable executable file. Timestamping is enabled by default and relies on the Comodo server (http://timestamp.comodoca.com/authenticode).

Since:
1.0
Author:
Emmanuel Bourg
